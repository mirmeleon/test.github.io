Commands
  • git status - dava statusa na fajlovete
  • git remote - pokazva remote repositories
  • git diff - pokazva samo ne indeksiranite promeni
  • git diff --staged -pokazva promenite v indexa (staged, demek indeksiranite)
  • git reset head file.txt - ako sme polzvali add file.txt tazi komanda shte premahne(revertne) advaneto v indexa
  • git rm file.txt - kazva na git da ne se zanimava s dadenia fajl (schita go che e iztrit)
  • git mv oldName.xt newName.txt - renamva file
  • git fetch - svalia danni ot remote repository ama ne pravi merge
  • git pull - svalia ama i mergva dannite i moje da se naloji da se razreshavat konflikti. Tova e = git fetch+git merge
  • git pull --rebase - druga strategia za merge ama slaga nashite promeni naj-otgore i taka e po-save
  • git checkout - smeniame branchove
  • git add file.txt ili git add . -dobavia fajlovete v indexa (stagva gi)
  • git commit -m "komentar" - dobavia fajlovete v head
  • git push - kachva gi v remote repoto
